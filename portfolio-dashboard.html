Dashboard Figma

Informations

  • Categorie: UX/UI
  • Date: Novembre 2023

Nous avons eu la chance de recevoir Ivan Dalmet de Bear studio pour nous apprendre les différents réflexes à avoir sur figma.
Pour ce faire, nous devions réaliser un dashboard destinné aux développeurs pour pouvoir déployer des sites web à partir d'un dépôt github.

Voici les consignes qui nous étaient assignées :
· Réaliser un écran tableau de bord sous Figma.
· Cible écran d’ordinateur portable 1440x900px.
· Le fichier doit utiliser des styles de couleurs.
· Le fichier doit utiliser des composants.
· Le fichier doit utiliser l’auto layout.
