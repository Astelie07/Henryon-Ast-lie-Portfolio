Ce projet est le projet final du BUT MMI.
Noous avons créé un jeu vidéo tournant autour du sujet des Backrooms, un univers horrifique très populaire sur internet, et réalisé tout une communication dans le but de promouvoir le jeu.
Le but de ce jeu est de s'échapper des Backrooms, tout en essayer de comprendre leurs origines.
Si vous souhaitez jouer au jeu, vous pouvez le télécharger via ce lien.
Je vous recommande également de regarder le trailer ci-contre ou encore de visiter le site internet dont le lien a été noté plus haut.

! Attention ce jeu est déconseillé aux personnes cardiaques !
